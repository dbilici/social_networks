## 🎯 Learning objectives
- Know what a social network is and how it models interconnected systems
- Know the fundamental theorems and measurements in graph theory
- Classify graphs and perform measurements
- Write **Python** code to analyze a network (`networkx` or `igraph`)
- Use **Pajek** to analyze a network
- Apply elementary graph-mining techniques
